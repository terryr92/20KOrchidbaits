# 20KOrchidbaits: Diagnostic SNP panels for orchid species and pollinaria genotyping

This repository contains all scripts, intermediate datasets, and derived resources used in the study:

**“Tracing species boundaries through target-capture genomic baits: Diagnostic SNP panels for orchid species and pollinaria genotyping”**

---

## Repository 

Raw sequencing data are not stored in this repository and will be made available through public sequence archives.

---

## Bait

The 20KOrchidbaits set includes:
- 20,000 nuclear baits
- 400 plastid baits
